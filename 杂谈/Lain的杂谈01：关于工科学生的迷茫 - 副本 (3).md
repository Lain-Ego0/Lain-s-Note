# 一、关于工科学生的迷茫
这是后台私信经常会提及的问题。由于我个人能力有限，部分天坑工科我可能无法给出解答，但机械、电子和计算机大类我可能会有一些看法。

1. 典型的版本T0（尤其是热衷于搞各种排行榜的）
    这是最让我头疼，也是最容易把自己玩死的一类人。他们的学习路线完全取决于营销号推什么。好玩的是现在前后端双非依然能找到大厂实习，嵌入式和新能源反而变成了真正的重学历，这么多年过去前后端依然是本科CS学生最好找到的工作。由于缺乏底层计算机体系结构或扎实的电路知识，他们只能停留在调包和跑通Demo的层面。企业招聘要的是能解决工程问题的熟练工或者有深度的专家，而不是一个背了一堆名词解释的行业观察者。

2. 象牙塔思维严重的绩点刷子
    这类迷茫通常发生在保研失败或者考研失利后。他们在学校里是好学生，每一门课都拿92/93+，奖学金拿到手软，但对工业界的技术栈一无所知。他们的迷茫在于：“我明明这么努力，为什么企业连面试机会都不给我？” 现实是，企业不是学校，不看你的考试能力。对于本科生尤其是双非，一个经典项目（比如苍穹外卖）或者多段的公司实习经历远比你的绩点更加重要。毕竟双非哪怕双非硕去学历厂在待遇上是不可能比得过92的，甚至有没有挂过科都不重要。

3. 把考研当成唯一的救命稻草
    这种迷茫带有很强的逃避色彩。很多人不是因为热爱科研去读研，而是因为不敢就业。 他们看到本科就业难，就理所当然地认为读个研就好了。于是大三开始切断所有实习机会，all in考研。残酷的真相是现在工科硕士尤其是双非工科硕的含金量也在极速稀释。很多时候，学历只是门票，能力才是饭碗，但很多人为了那张门票把做饭的手艺都丢了。

4. 陷入完美技术栈的死循环
    这一类主要集中在计算机和电子类学生中，尤其是对于工业届有不切实际幻想的人，他们总是试图寻找一种学了就能一劳永逸的神仙技术。他们在选择语言和方向上花费了巨大的时间成本，甚至存在一种特殊的洁癖（点名Rust）。工科没有完美的技术栈，只有适合当下的解决方案。 任何一门主流技术学精了都能混口饭吃，最怕的就是在犹豫中蹉跎了最宝贵的大学时光。

总而言之，目前的就业环境确实恶劣，但这并不意味着没有机会。真正的机会属于那些不盲目跟风、不迷信学历、并能尽早接触工业界真实需求的人。他们往往会成为行业的中流砥柱，或者是创业者。这是大多数学校不存在的条件，也是我一直尽力寻找校外资源的原因。
